-- İlk dəfə repository yaradanda , əsasən özü bildirir bu kodları , git add . etməyi ancaq unutmayın --
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin -- ilk dəfə repo yaradanda verdiyi remote ünvan
git push -u origin main
